# 🚲 서울시 공공자전거(따릉이) 데이터 분석 프로젝트

본 프로젝트는 **서울시 공공자전거 이용 데이터(bic.csv)**를 기반으로,\
대여소별 이용 패턴을 분석하고 시각화하는 Streamlit 웹 대시보드입니다.

------------------------------------------------------------------------

## 📌 프로젝트 목적

-   서울시 따릉이 이용 현황 파악\
-   대여소별 인기/비인기 대여소 분석\
-   이용 건수, 이용 시간, 이동 거리 등 주요 지표 분석\
-   사용자 기준(성별/연령대)별 이용 패턴 분석\
-   Streamlit 기반 대시보드 제작

------------------------------------------------------------------------

## 🛠 사용 기술 스택

  분야          기술
  ------------- ---------------------------------------------

## 📊 기능 소개 상세

### 1️⃣ 메인 페이지

-   서울시 공공자전거 데이터 소개\
-   전체 데이터 미리보기\
-   주요 분석 목표 설명

------------------------------------------------------------------------

### 2️⃣ 대여소별 데이터 보기

특정 대여소 선택 시,\
해당 대여소의 전체 이용 기록만 필터링하여 제공합니다.

------------------------------------------------------------------------

### 3️⃣ 대여소별 이용건수 TOP10

-   groupby 활용하여 대여소별 이용건수 합산\
-   상위 10개 대여소 목록 및 막대그래프 시각화\
-   이용건수가 가장 많은 대여소 확인 가능

예시 결과:\
\> 마곡나루역 2번 출구 --- 이용자 최다

------------------------------------------------------------------------

### 4️⃣ 대여소별 총 이용시간 TOP10

-   `이용시간(분)` 합계를 기준으로 상위 10개 대여소 분석\
-   주로 한강공원 근처 대여소가 높은 이용시간을 보임\
-   그래프를 통한 직관적 확인 가능

------------------------------------------------------------------------

### 5️⃣ 데이터 선택 및 통계 기능

사용자가 직접 기준 선택:

  항목        선택 옵션
  ----------- ---------------------------------------
  그룹 기준   대여소명 / 연령대코드 / 성별
  분석 항목   이용건수 / 이동거리(M) / 이용시간(분)
  통계 방식   sum / mean / max / min / count

선택한 기준에 대해 상위 10개 항목을 정렬해 시각화함.
