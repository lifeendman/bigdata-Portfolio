# 🚲 서울시 공공자전거(따릉이) 데이터 분석 프로젝트

## 📌 데이터분석 프로젝트 설명

본 프로젝트는 서울시 공공자전거 **따릉이** 데이터를 활용해 이용 행태를
분석하고, 이를 Streamlit 기반 대시보드로 시각화하는 데이터 분석
프로젝트입니다.

------------------------------------------------------------------------

## 📘 주제

**서울시 공공자전거(따릉이) 이용 패턴 분석 및 시각화**

------------------------------------------------------------------------

## 🎯 주제 선택 이유

-   시민들이 일상생활에서 가장 많이 사용하는 공유 이동수단 중 하나이기
    때문
-   공공데이터가 잘 정리되어 있어 다양한 분석이 가능
-   대여소별 이용량, 이용 시간 등의 패턴을 통해 도시 교통 활용 흐름을
    이해할 수 있음
-   실제 생활 속 데이터 기반 프로젝트로 실용성과 의미가 큼

------------------------------------------------------------------------

## 📊 데이터 분석 내용

### 1️⃣ 대여소별 이용건수 분석

-   이용건수 합계를 기준으로 상위 10개 대여소 분석
-   시각화(막대 그래프)를 통한 인기 대여소 파악

### 2️⃣ 대여소별 이용시간 분석

-   대여소별 총 이용시간(분)을 기준으로 TOP10 분석
-   한강공원 인근 대여소들이 긴 이용시간을 보임

### 3️⃣ 특정 대여소 데이터 상세 조회

-   선택한 대여소의 모든 이용 기록을 필터링하여 제공

### 4️⃣ 사용자 기준별 분석

-   성별, 연령대코드 등을 기준으로
    -   이용건수
    -   이동거리
    -   이용시간
        에 대한 통계(sum, mean, max, count) 계산

### 5️⃣ Streamlit 기반 데이터 시각화

-   plotly 그래프 활용
-   선택형 메뉴로 사용자 탐색 기능 구현
